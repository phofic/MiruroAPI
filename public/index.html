Streaming in 3 Steps

Get M3U8 streaming URLs for any anime episode in 3 simple API calls.

Get Episodes

Fetch all episodes from multiple providers

GET /api/episodes/{anilistId}

Get Sources

Get M3U8 URL from episode ID

GET /api/watch/{provider}/{id}/{cat}/{slug}

Play

Feed M3U8 URL to any HLS player

Quick Start

Get started in seconds. No API key required.

JavaScript — Fetch search results
const res = await fetch("https://mirurotvapi.vercel.app/api/search?query=naruto");
const { success, results } = await res.json();
console.log(results.total); // 5000+
console.log(results.results[0].title.romaji); // "NARUTO"
JavaScript — Get streaming sources
// Step 1: Get episodes
const eps = await fetch("https://mirurotvapi.vercel.app/api/episodes/20");
const { results } = await eps.json();
const firstEp = results.providers.kiwi.episodes.sub[0];

// Step 2: Get M3U8 URL
const stream = await fetch(`https://mirurotvapi.vercel.app${firstEp.id}`);
const { results: sources } = await stream.json();
console.log(sources.streams[0].url); // "https://.../master.m3u8"
cURL — Quick test
curl "https://mirurotvapi.vercel.app/api/search?query=one+piece&per_page=3"
